HTML Basics

Elementos aprendidos en
programación web en español

DOCTYPE

La declaración <!DOCTYPE> le indica al navegador qué versión de HTML se está utilizando. En HTML5 se usa <!DOCTYPE html> y asegura que el navegador interprete el código correctamente. 


  <!DOCTYPE html>

HTML

La etiqueta <html> es la raíz de un documento HTML. Contiene todo el contenido de la página: cabecera, cuerpo, scripts y más. 


  <html> </html>

Head

La etiqueta <head> contiene metadatos: título de la página, enlaces a hojas de estilo, scripts o descripciones. Estos datos no se muestran directamente al usuario. 


  <head>
    <title>Mi página</title>
    <link rel="stylesheet" href="style.css">
  </head>

Body

La etiqueta <body> contiene todo el contenido visible para el usuario: texto, imágenes, videos, formularios, enlaces y mucho más. 


  <body>
    <h1>Hola Mundo</h1>
    <p>Este es mi primer párrafo.</p>
  </body>

Title

La etiqueta <title> define el título de la pestaña del navegador. También influye en el SEO (posicionamiento en buscadores). 


  <title>Mi primera página HTML</title>

H1

La etiqueta <h1> representa el encabezado de primer nivel, el más importante en la jerarquía de títulos. Se recomienda usar solo uno por página para mantener una estructura semántica adecuada y mejorar el SEO. 


  <h1>Título Principal de la Página</h1>

H2

La etiqueta <h2> representa el encabezado de segundo nivel. Se utiliza para dividir el contenido en secciones principales que están subordinadas al título principal (h1). Puede haber múltiples elementos h2 en una página. 


  <h2>Sección Principal del Contenido</h2>

H3

La etiqueta <h3> representa el encabezado de tercer nivel. Se utiliza para subsecciones dentro de una sección h2. Ayuda a crear una estructura jerárquica clara del contenido, mejorando la legibilidad y el SEO. 


  <h3>Subsección de Contenido</h3>

H4

La etiqueta <h4> representa el encabezado de cuarto nivel. Se utiliza para subdivisiones dentro de una sección h3. Aunque es menos común, es útil para organizar contenido muy detallado o técnico. 


  <h4>Detalles Específicos del Tema</h4>

H5

La etiqueta <h5> representa el encabezado de quinto nivel. Se utiliza para subdivisiones muy específicas dentro de una sección h4. Su uso es poco frecuente, pero puede ser necesario en documentos muy extensos o técnicos. 


  <h5>Información Adicional Especializada</h5>

H6

La etiqueta <h6> representa el encabezado de sexto nivel, el menos importante en la jerarquía. Se utiliza para subdivisiones extremadamente específicas. Es raro necesitar este nivel de detalle en la mayoría de las páginas web. 


  <h6>Detalle Mínimo o Nota al Pie</h6>

P

La etiqueta <p> se utiliza para definir un párrafo de texto. Es una de las etiquetas más usadas en HTML. P significa paragraph (párrafo). 


  <p>Este es un párrafo en HTML.</p>

Span

La etiqueta <span> es un contenedor en línea genérico que se utiliza para aplicar estilos o manipular porciones específicas de texto dentro de un elemento más grande. A diferencia de <div>, <span> no crea saltos de línea. 


  <p>Este es un texto con una <span style="color: red;">palabra en rojo</span> usando span.</p>

A

La etiqueta <a> (anchor o ancla) crea hipervínculos a otras páginas web, archivos, ubicaciones dentro del mismo documento, o cualquier otra URL. Es fundamental para la navegación web. 


  <a href="https://www.ejemplo.com" target="_blank">Visitar Ejemplo.com</a>

OL

La etiqueta <ol> (Ordered List) crea una lista ordenada donde los elementos se muestran con números o letras en secuencia. Es ideal para presentar pasos, instrucciones o cualquier contenido que requiera un orden específico. 


  <ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
    <li>Tercer elemento</li>
  </ol>

UL

La etiqueta <ul> (Unordered List) crea una lista no ordenada donde los elementos se muestran con viñetas. Se utiliza para presentar elementos que no requieren un orden específico o una secuencia. 


  <ul>
    <li>Elemento de lista</li>
    <li>Otro elemento</li>
    <li>Más elementos</li>
  </ul>

DL

La etiqueta <dl> (Description List) crea una lista de descripciones que consiste en términos y sus definiciones. Se utiliza comúnmente para glosarios, listas de metadatos o para mostrar pares de término-definición. 


  <dl>
    <dt>HTML</dt>
    <dd>Lenguaje de marcado para crear páginas web.</dd>
    <dt>CSS</dt>
    <dd>Lenguaje para definir el estilo de documentos HTML.</dd>
  </dl>

LI

La etiqueta <li> (List Item) define un elemento dentro de una lista, ya sea ordenada (<ol>) o no ordenada (<ul>). Cada elemento de la lista debe estar envuelto en esta etiqueta. 


  <ul>
    <li>Primer elemento de la lista</li>
    <li>Segundo elemento de la lista</li>
  </ul>

DT

La etiqueta <dt> (Description Term) define un término en una lista de descripción (<dl>). Se utiliza junto con <dd> para crear pares de término-definición en listas de descripción. 


  <dl>
    <dt>HTTP</dt>
    <dd>Protocolo de transferencia de hipertexto.</dd>
  </dl>

DD

La etiqueta <dd> (Description Details) define la descripción o definición de un término en una lista de descripción (<dl>). Siempre va seguido de un <dt> que define el término que se está describiendo. 


  <dl>
    <dt>CSS</dt>
    <dd>Hojas de estilo en cascada, usadas para estilizar páginas web.</dd>
  </dl>

Table

La etiqueta <table> crea una tabla para mostrar datos tabulares. Dentro de una tabla, se utilizan filas (<tr>) y celdas (<td> o <th>) para organizar la información de manera estructurada. 


  <table>
    <tr>
      <th>Nombre</th>
      <th>Edad</th>
    </tr>
    <tr>
      <td>María</td>
      <td>25</td>
    </tr>
  </table>

TR

La etiqueta <tr> (Table Row) define una fila dentro de una tabla HTML. Cada fila contiene una o más celdas, que pueden ser de encabezado (<th>) o de datos (<td>). 


  <table>
    <tr>
      <th>Producto</th>
      <th>Precio</th>
    </tr>
    <tr>
      <td>Laptop</td>
      <td>$800</td>
    </tr>
  </table>

TD

La etiqueta <td> (Table Data) define una celda estándar en una tabla HTML que contiene datos. Estas celdas se encuentran dentro de las filas (<tr>) y representan el contenido regular de la tabla. 


  <table>
    <tr>
      <td>Dato 1</td>
      <td>Dato 2</td>
    </tr>
  </table>

TH

La etiqueta <th> (Table Header) define una celda de encabezado en una tabla HTML. Estas celdas se utilizan para describir el contenido de las columnas o filas y normalmente se muestran en negrita y centradas. 


  <table>
    <tr>
      <th>Nombre</th>
      <th>Apellido</th>
    </tr>
    <tr>
      <td>Juan</td>
      <td>Pérez</td>
    </tr>
  </table>

Thead

La etiqueta <thead> agrupa el contenido del encabezado en una tabla HTML. Contiene las filas de encabezado (<tr> con <th>) y ayuda a diferenciar semanticamente el encabezado del cuerpo de la tabla, mejorando la accesibilidad y el estilo. 


  <table>
    <thead>
      <tr>
        <th>Mes</th>
        <th>Ventas</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Enero</td>
        <td>$1000</td>
      </tr>
    </tbody>
  </table>

Tbody

La etiqueta <tbody> agrupa el contenido principal del cuerpo de una tabla HTML. Contiene las filas de datos de la tabla y se utiliza junto con <thead> y <tfoot> para estructurar semanticamente la tabla. 


  <table>
    <thead>
      <tr>
        <th>Producto</th>
        <th>Precio</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Laptop</td>
        <td>$800</td>
      </tr>
      <tr>
        <td>Mouse</td>
        <td>$20</td>
      </tr>
    </tbody>
  </table>

Tfoot

La etiqueta <tfoot> agrupa el contenido del pie de una tabla HTML. Normalmente contiene resúmenes, totales o notas al pie de la tabla. Aunque se coloca en el código después de <thead>, el navegador lo muestra al final de la tabla. 


  <table>
    <thead>
      <tr>
        <th>Producto</th>
        <th>Precio</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Laptop</td>
        <td>$800</td>
      </tr>
      <tr>
        <td>Mouse</td>
        <td>$20</td>
      </tr>
    </tbody>
    <tfoot>
      <tr>
        <td>Total</td>
        <td>$820</td>
      </tr>
    </tfoot>
  </table>

Header

La etiqueta <header> representa la cabecera de una página o sección. Generalmente contiene el logo, menús de navegación o títulos principales. 


  <header>
    <h1>Bienvenido a mi sitio</h1>
  </header>

Nav

La etiqueta <nav> se usa para definir un menú de navegación en la página. Suele contener enlaces <a> hacia distintas secciones. Nav significa Navigation (Navegación) 


  <nav>
    <ul>
      <li><a href="#home">Inicio</a></li>
      <li><a href="#about">Acerca de</a></li>
      <li><a href="#contact">Contacto</a></li>
    </ul>
  </nav>

Main

La etiqueta <main> indica el contenido principal de la página. Solo debe existir una por documento. 


  <main>
    <p>Este es el contenido principal.</p>
  </main>

Aside

La etiqueta <aside> se usa para contenido adicional o relacionado, como barras laterales, publicidad o enlaces extra. 


  <aside>
    <p>Publicidad o enlaces relacionados</p>
  </aside>

Footer

La etiqueta <footer> define el pie de página, donde suele colocarse información de contacto, derechos de autor o enlaces a políticas. 


  <footer>
    <p>© 2024 Mi sitio web</p>
  </footer>

Img

Muestra una imagen en la página. Usa el atributo src para indicar la ruta. 


  <img src="img/Profile.jpg" alt="Ejemplo de imagen">

Video

Permite incrustar un video en la página. Se puede controlar con el atributo controls. 


  <video controls width="320" src="video/ejemplo.mp4"></video>

Source

La etiqueta <source> se utiliza dentro de elementos multimedia como <video>, <audio> o <picture> para especificar múltiples recursos multimedia. Esto permite ofrecer diferentes formatos para garantizar la compatibilidad con diversos navegadores. 


  <video controls width="320">
    <source src="video/ejemplo.mp4" type="video/mp4">
    <source src="video/ejemplo.webm" type="video/webm">
    Tu navegador no soporta el elemento de video.
  </video>

Audio

Reproduce archivos de audio. Se recomienda el formato MP3 para compatibilidad. 


  <audio src="audio/ejemplo.mp3" controls></audio>

Iframe

La etiqueta <iframe> (Inline Frame) incrusta otro documento HTML dentro del documento actual. Se utiliza comúnmente para incorporar contenido externo como videos de YouTube, mapas de Google, o otras páginas web dentro de una página. 


  <iframe 
    src="https://www.example.com" 
    width="600" 
    height="400" 
    title="Ejemplo de iframe">
  </iframe>

Object

La etiqueta <object> define un contenedor para un recurso externo, que puede ser una página web, una imagen, un plugin multimedia o cualquier otro tipo de contenido. Es más versátil que <iframe> pero menos utilizado en la práctica moderna. 


  <object data="documento.pdf" type="application/pdf" width="600" height="400">
    <p>Tu navegador no soporta el elemento object.</p>
  </object>

Form

La etiqueta <form> se utiliza para crear un formulario que permite al usuario enviar información al servidor. Puede contener inputs, selects, textareas, botones y otros elementos de entrada. Sus atributos más importantes son action (la URL que recibirá los datos) y method (GET o POST). 


<form action="/enviar" method="post">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre">

  <input type="submit" value="Enviar">
</form>

Input

La etiqueta <input> define un campo interactivo donde el usuario puede introducir datos. Existen muchos tipos de input según la información que se necesite: text, password, email, number, date, color, file, checkbox, radio, range, url, entre otros. Además puede tener atributos como placeholder, required, maxlength, min y max. 


<!-- Text input -->
<input type="text" name="usuario" placeholder="Ingresa tu usuario" required>

<!-- Password input -->
<input type="password" name="clave" placeholder="Contraseña" required>

<!-- Email input -->
<input type="email" name="correo" placeholder="correo@ejemplo.com">

<!-- Number input con rango -->
<input type="number" name="edad" min="1" max="120">

<!-- Checkbox y Radio -->
<input type="checkbox" name="suscripcion"> Suscribirse
<input type="radio" name="genero" value="masculino"> Masculino
<input type="radio" name="genero" value="femenino"> Femenino

Select

La etiqueta <select> crea un menú desplegable que permite al usuario elegir una opción de varias disponibles. Se utiliza junto con la etiqueta <option> para definir cada opción. Puede tener atributos como name, multiple (permite seleccionar varias opciones) y required. 


<label for="pais">País:</label>
<select name="pais" id="pais" required>
  <option value="" disabled selected>Selecciona tu país</option>
  <option value="mx">México</option>
  <option value="es">España</option>
  <option value="ar">Argentina</option>
</select>

Textarea

La etiqueta <textarea> define un área de texto de varias líneas donde el usuario puede introducir información más extensa que en un input normal. Sus atributos importantes incluyen name, rows (filas visibles), cols (columnas visibles), placeholder y maxlength. 


<label for="comentario">Comentario:</label>
<textarea id="comentario" name="comentario" rows="5" cols="40" placeholder="Escribe tu comentario aquí..." required></textarea>

Label

La etiqueta <label> se utiliza para definir un rótulo para un elemento de formulario, como un input o textarea. Su atributo más importante es for, que debe coincidir con el id del elemento que describe. Esto mejora la accesibilidad y permite que al hacer clic en el label se active el input correspondiente. 


<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">

<label for="comentario">Comentario:</label>
<textarea id="comentario" name="comentario"></textarea>
